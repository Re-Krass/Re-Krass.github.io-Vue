# re-krass-website

## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run serve
```

### Compiles and minifies for production
```
pnpm run build
```

### Run your tests
```
pnpm run test
```

### Lints and fixes files
```
pnpm run lint
```

```bash

```
