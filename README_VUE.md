```
pnpm install
```

```

```

```
pnpm run build
```

### Run your tests

pnpm run test
```

```

```

## Docsify Documentation

docsify init ..\docs\

# preview site
docsify serve ..\docs\
```

Preview server [Server](http://localhost:3000)
